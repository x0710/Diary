# Diary

一个使用 **Rust** 编写的极简个人日记命令行工具。

> 本项目仍在持续开发中。

---

## 项目简介

**Diary** 是一个轻量级的命令行日记工具，用于快速记录、查看和管理每天的简要笔记。

设计目标：

* 简单、快速
* 适合脚本和终端使用
* 使用 SQLite 本地存储
* 易于扩展和重构

项目仍在演进中，部分功能可能会发生变化。

---

## 功能特性

* ✅ 按日期添加日记
* ✅ 查看指定日期的记录
* ✅ 使用 SQLite 持久化存储
* ✅ 列出已有记录

## 安装

### 从源码构建

```bash
git clone https://github.com/x0710/diary.git
cd diary
cargo build --release
```

生成的可执行文件位于：

```text
target/release/diary
```

你可以将其移动到 `$PATH` 中方便使用。

### 使用发布版本

可直接从 GitHub Releases 页面下载已编译好的二进制文件。

---

## 使用方法

运行 `diary` 进入交互模式：

```bash
diary
```

### 添加一条记录

```bash
>: add 2025-01-01 "今天发生的事情"
```

支持的日期格式：

* `YYYY-MM-DD`
* `YYYYMMDD`
* `today` / `t`（今天）
* `tomorrow` / `m`（明天）
* `yesterday` / `y`（昨天）

### 查看某天的记录

```bash
>: check 2025-01-01
```

### 列出所有记录

```bash
>: ls
```

---

## 数据存储

* 所有数据均保存在本地 SQLite 数据库中
* 不联网
* 不启动后台服务
